Discord Social Graph

Carte | Confidentialité | Tutoriel

Comment ça marche ?

Étape 1 : Connexion et Analyse

  1. Cliquez sur le bouton "Se connecter via Discord" sur la page d'accueil.
  2. Discord vous demandera d'autoriser l'application à voir votre profil et vos serveurs.
  3. Une fois autorisé, l'application récupère votre liste de serveurs et stocke ces informations de manière permanente dans la base de données.

Étape 2 : Construction du Graphe

Chaque utilisateur qui se connecte ajoute sa propre "carte" au système. Le graphe combine toutes ces cartes :

  • Les cercles représentent les utilisateurs.
  • Les rectangles représentent les serveurs Discord.
  • Les liens gris représentent l'appartenance d'un utilisateur à un serveur.
  • Les liens rouges et épais (liens "serveur_commun") apparaissent entre deux utilisateurs qui partagent un ou plusieurs serveurs. Plus le lien est épais, plus vous partagez de serveurs.

Étape 3 : Utilisation du Graphe

  • Déplacement : Vous pouvez glisser-déposer les nœuds pour mieux organiser la visualisation.
  • Détails : Cliquez sur n'importe quel cercle ou rectangle pour afficher l'ID et les détails dans le panneau mobile de droite.
  • Réinitialisation : Le bouton "Réinitialiser la Carte Locale" efface uniquement l'affichage actuel dans votre navigateur. Les données stockées dans la base de données restent intactes pour la prochaine session.

Discord Social Graph

Carte | Confidentialité | Tutoriel

Que faisons-nous de vos données Discord ?

Cette application est conçue pour être transparente sur les données collectées et stockées. Elle nécessite votre autorisation pour lire certaines informations de base sur votre compte Discord.

1. Les Données que nous Collectons (Scope)

Lors de la connexion, nous demandons l'accès aux informations suivantes :

  • `identify` : Votre nom d'utilisateur, votre ID utilisateur (unique), votre avatar.
  • `guilds` : La liste des serveurs Discord dont vous êtes membre.

2. Les Données que nous Stockons (Base de Données)

Afin de construire le graphe social persistant, nous stockons de manière sécurisée dans notre base de données MongoDB :

  • Votre **ID utilisateur** (clé unique de la carte).
  • Votre **nom d'utilisateur** et une couleur/type pour l'affichage.
  • Les **IDs et noms des serveurs** que vous partagez.
  • Les **relations** entre vous et ces serveurs.

3. Les Données que nous NE collectons PAS

Nous n'avons **PAS** accès aux informations suivantes et nous ne les stockons jamais :

  • Vos messages privés ou messages de serveurs.
  • Votre adresse e-mail ou mot de passe Discord.
  • Votre jeton d'accès (il est utilisé une seule fois par notre proxy Render, puis jeté).

But : Les données sont uniquement utilisées pour visualiser votre réseau social Discord et les points communs avec d'autres utilisateurs qui se connectent à l'application. Elles ne sont pas partagées ou vendues à des tiers.

4. Demande de Suppression

Si vous souhaitez que vos données soient définitivement retirées de notre base de données MongoDB, veuillez contacter l'administrateur du projet via le Discord de : Banou999.
